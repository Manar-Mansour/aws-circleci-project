##Pipeline process
1- Spin up environment 
2- Preparing the environment variables
3- Checkout code
4- Install Node.js
5- Install AWS CLI - latest
6- Configure AWS Access Key ID
7- Setting up Elastic Beanstalk CLI
8- Front-End Install 
9- Back-End Install
10- Front-End Build
11- Back-End Build
12- Deploy API
13- Deploy App
